# Projektname
Flappy Bird

## Kurzbeschreibung
Kleines Jump and Run Game mit Highscore.

## Features
- Um den Bird gegen die Schwerkraft am Fliegen zu halten: Leertaste
- Pausieren: p
- menu.py : im Terminal Vogel und Spielgeschwindigkeit wählen

## Setup
- Repository klonen
- im Terminal in den Ordner flappy_bird navigieren
- flappy_bird.py starten

## Nutzung
- zum Fliegen: Leertaste drücken
- Ziel: Nicht die Röhren, den Boden oder die Decke berühren

## Struktur
Kurz, wenn nötig:
- flappy_bird.py – Quellcode
- menu.py – Wahl von Vogel und Geschwindigkeit (nicht nötig)
- images - für bird, pipes und background

## Status
- PoC

## Hinweis
Dieses Projekt ist eine Nachprogrammierung von Flappy Bird und dient ausschließlich zu Lernzwecken.
